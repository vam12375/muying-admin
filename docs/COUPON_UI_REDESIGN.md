# 优惠券列表UI重新设计

## 设计目标

基于用户反馈"页面UI不够美观，操作那里太丑了"，对优惠券列表页面进行全面UI重构。

## 设计原则

遵循 **KISS, YAGNI, SOLID** 原则：
- **KISS**: 保持界面简洁直观，减少视觉噪音
- **YAGNI**: 只实现必要的功能，避免过度设计
- **SOLID**: 组件结构清晰，易于维护和扩展

## 主要改进

### 1. 布局变更：从表格到卡片

**之前**：传统表格布局
- 信息密集，难以快速浏览
- 操作按钮隐藏在下拉菜单中
- 视觉层次不清晰

**现在**：现代卡片式布局
- 每个优惠券独立卡片展示
- 信息分组清晰，易于扫描
- 操作按钮直接可见

### 2. 优惠券视觉卡片

新增优惠券视觉化展示：
- **固定金额券**：蓝色渐变背景
- **折扣券**：绿色渐变背景
- 显示优惠金额/折扣
- 显示使用门槛
- 装饰性圆圈增强优惠券感

### 3. 操作按钮优化

**之前**：
- 所有操作隐藏在"..."下拉菜单中
- 需要点击才能看到可用操作
- 交互路径长

**现在**：
- 4个主要操作按钮直接展示
- 每个按钮有独特的hover颜色：
  - 编辑：蓝色
  - 复制：紫色
  - 启用/停用：绿色/橙色
  - 删除：红色
- 图标+文字，语义清晰

### 4. 统计信息可视化

新增4个统计卡片：
- 已领取数量
- 已使用数量
- 使用率百分比
- 剩余数量

每个卡片独立背景，信息一目了然。

### 5. 批量操作优化

- 选中项时显示醒目的操作栏
- 渐变背景突出显示
- 显示选中数量
- 提供取消选择和批量删除操作

### 6. 动画效果

- 卡片入场动画（渐入+上移）
- 批量操作栏动画
- 卡片hover阴影变化
- 按钮hover颜色过渡

## 技术实现

### 组件结构

```tsx
<div className="space-y-4">
  {/* 批量操作栏 */}
  {selectedIds.length > 0 && <BatchOperationBar />}
  
  {/* 卡片列表 */}
  <div className="grid grid-cols-1 gap-4">
    {coupons.map(coupon => (
      <CouponCard key={coupon.id}>
        <Checkbox />
        <VisualCoupon />
        <CouponInfo />
        <ActionButtons />
      </CouponCard>
    ))}
  </div>
  
  {/* 分页 */}
  <Pagination />
</div>
```

### 样式系统

- 使用 Tailwind CSS 实用类
- 深色模式完全支持
- 响应式设计（移动端友好）
- 渐变色系统：
  - 粉色到紫色：主题色
  - 蓝色：固定金额券
  - 绿色：折扣券

### 动画库

使用 Framer Motion：
- `initial`: 初始状态
- `animate`: 动画目标状态
- `transition`: 动画时序控制

## 用户体验提升

1. **视觉层次清晰**：卡片布局自然分组信息
2. **操作效率提升**：常用操作一键可达
3. **信息密度优化**：重要信息突出显示
4. **交互反馈丰富**：hover效果、动画、颜色变化
5. **移动端友好**：响应式布局适配各种屏幕

## 性能考虑

- 使用 `motion.div` 的 `transition.delay` 实现错峰动画
- 避免过度动画影响性能
- 保持组件结构简单，易于渲染
