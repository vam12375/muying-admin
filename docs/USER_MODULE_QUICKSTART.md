# 用户管理模块快速启动指南

## 快速开始

### 1. 访问用户管理

在浏览器中访问：`http://localhost:3000/users`

或通过左侧导航菜单点击"用户管理"。

### 2. 查看用户列表

用户列表展示所有用户账户信息：
- 用户ID、用户名、昵称
- 联系方式（邮箱、手机号）
- 账户余额、累计充值、累计消费
- 账户状态（正常/冻结）
- 创建时间

### 3. 搜索用户

在搜索框中输入：
- 用户名
- 昵称
- 邮箱
- 手机号

点击"查询"按钮或按回车键搜索。

### 4. 筛选用户

使用"状态"下拉框筛选：
- 全部：显示所有用户
- 正常：只显示正常状态的用户
- 冻结：只显示冻结状态的用户

### 5. 用户充值

1. 找到目标用户
2. 点击操作列的"..."按钮
3. 选择"充值"
4. 填写充值信息：
   - 充值金额（必填）
   - 支付方式（必选）
   - 充值说明（可选）
   - 备注（可选）
5. 点击"确认充值"

### 6. 调整余额

1. 找到目标用户
2. 点击操作列的"..."按钮
3. 选择"调整余额"
4. 选择调整类型：
   - 增加：增加用户余额
   - 减少：减少用户余额
5. 输入调整金额
6. 填写调整原因（必填）
7. 点击"确认调整"

**注意**：减少余额时，系统会检查用户余额是否充足。

### 7. 查看交易历史

1. 找到目标用户
2. 点击操作列的"..."按钮
3. 选择"交易历史"
4. 查看用户的所有交易记录：
   - 账户摘要（当前余额、累计充值、累计消费）
   - 交易记录列表
   - 交易详情（流水号、类型、金额、状态等）
5. 使用分页浏览更多记录
6. 点击"刷新"更新数据

### 8. 冻结/解冻账户

1. 找到目标用户
2. 点击操作列的"..."按钮
3. 选择"冻结账户"或"解冻账户"
4. 确认操作

**注意**：冻结账户后，用户将无法进行任何交易操作。

## 常见问题

### Q1: 充值失败怎么办？

**A**: 检查以下几点：
1. 充值金额是否大于0
2. 是否选择了支付方式
3. 网络连接是否正常
4. 后端服务是否正常运行

### Q2: 调整余额时提示余额不足？

**A**: 减少余额时，调整金额不能大于用户当前余额。请检查用户当前余额，或选择"增加"类型。

### Q3: 交易历史加载失败？

**A**: 检查以下几点：
1. 网络连接是否正常
2. 后端服务是否正常运行
3. 用户ID是否正确
4. 尝试点击"刷新"按钮重新加载

### Q4: 冻结账户后用户还能登录吗？

**A**: 冻结账户只影响用户的交易功能，不影响登录。如需禁止用户登录，请联系系统管理员。

### Q5: 如何查看某个用户的详细信息？

**A**: 目前可以通过以下方式查看：
1. 在用户列表中查看基本信息
2. 点击"交易历史"查看账户摘要和交易记录
3. 后续版本将添加用户详情页面
